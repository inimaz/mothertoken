## Design Context

### Users
Developers, AI researchers, and engineers comparing language model tokenization efficiency and costs.

### Brand Personality
Elegant, minimal, modern.

### Aesthetic Direction
Refined minimalist and data-friendly. Inspired by sleek, developer-focused tools like `canirun.ai` and `skills.sh`. High-contrast, clean typography, precise spacing, and subtle premium accents.

### Design Principles
1. **Clarity over decoration:** Data is the primary focus; visual elements should enhance readability, not distract.
2. **Precision and polish:** Use subtle micro-interactions, fluid sizing, and refined spacing to elevate the experience from a raw tool to a premium product.
3. **Elegant data visualization:** Present dense metrics cleanly using purposeful alignment, subtle color coding, and typography hierarchy.
